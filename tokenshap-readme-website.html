TokenSHAP

A Novel Approach to Token Importance in Text Classification

View on GitHub

Overview

TokenSHAP is a novel method for integrating token importance into text classification explanations. It enhances traditional SHAP values by considering the contextual significance of individual tokens, providing more accurate and interpretable results for text classification tasks.

Methodology

Token Importance Calculation

Our approach combines:

  • Token-level SHAP values
  • Contextual token importance
  • Integration with transformer-based models

Implementation Steps

  1. Initialize TokenSHAP with your pre-trained model
  2. Generate token-level importance scores
  3. Calculate integrated SHAP values
  4. Visualize the results

Installation

pip install git+https://github.com/ronigold/TokenSHAP.git

Usage Example

import torch from transformers import AutoModelForSequenceClassification, AutoTokenizer from tokenshap import TokenSHAP # Load your model and tokenizer model = AutoModelForSequenceClassification.from_pretrained('bert-base-uncased') tokenizer = AutoTokenizer.from_pretrained('bert-base-uncased') # Initialize TokenSHAP explainer = TokenSHAP(model, tokenizer) # Generate explanations text = "Your input text here" explanation = explainer.explain(text) # Visualize results explainer.visualize(explanation)
